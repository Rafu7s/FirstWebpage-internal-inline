```
#top>#logo>#menu-top->#main>#left,#body>h1,#content->#footer
```

- Internal
```
<style>...</style> (w head)
```
- Head
```
<link rel="stylesheet" href="*.css">
<script src="*.js" type="..."></script>
<meta charset/name(decription/keywords)>
<title>
```

```
<img>, <hr>, <br>, <video>, <audio>, <script>
komentarz: <!--...-->
```
